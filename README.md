# modelagem-arquiteturas-escalaveis

Este repositório contém a documentação e os arquivos técnicos do projeto de iniciação científica voltado à modelagem, implementação e avaliação experimental de arquiteturas de software para sistemas web escaláveis.

## 🎯 Objetivo

Desenvolver e avaliar protótipos arquiteturais (monolito modular vs microserviços/serverless, por exemplo) para comparar escalabilidade, desempenho e custo.
